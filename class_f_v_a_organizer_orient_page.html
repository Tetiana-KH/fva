Detailed Description

FVAOrganizerOrientPage is a child of QWizardPage and implements the next UI functions:

  1. QLabel to suggest a user to check orintation;
  2. QPushButton to start "/jpegr/jpegr.exe" to check orintations.

Showing the elements and flow is based on class FvaConfiguration. This class implements "View" functions from MVC pattern.
